# uxmDragonEvent Wiki

In this repository, you can find default configuration files of latest uxmDragonEvent version and also a wiki for learning everything related to uxmDragonEvent.

## Adding New Language Support

You can find language strings inside config.yml, copy the config file and translate everything inside it then create a PR including the new translated files, you can name it ``config_<language>.yml``. Our staff will check and merge your work if its suitable.

Click [here](https://github.com/UXPLIMA/uxmDragonEvent-wiki/wiki) to reach the wiki.
